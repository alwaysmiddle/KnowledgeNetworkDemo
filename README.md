# Knowledge Network Demo

Proof-of-concept frontend demos for the Knowledge Network master's thesis project.

## Features

- **Multi-Layer Knowledge Graph** - Dynamic layer construction and navigation

See `KnowledgeNetwork-design-docs/features/` for detailed specifications.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── app/             # App shell, layout, routing
├── features/        # Isolated demo implementations
└── shared/          # Minimal shared utilities
```
